<h1>1.6 Procesamiento de mapas de bits</h1>
<p>Un mapa de bits (bitmap) es una representación de imagen basada en píxeles.</p>

<h2>Características</h2>
<ul>
<li>Matriz de píxeles</li>
<li>Dependiente de resolución</li>
<li>Cada píxel almacena color</li>
</ul>
<img width="310" height="247" alt="image" src="https://github.com/user-attachments/assets/e898836b-0aca-44e9-9b42-e2a45a0a7d03" />

<h2>Resolución</h2>
<p>Se expresa como:</p>
<p>ancho×alto</p>
<p>Ejemplo:</p>
<ul>
<li>1920 × 1080 </li>
</ul>
<p>Mayor resolución → mayor detalle → mayor tamaño. </p>
<img width="308" height="246" alt="image" src="https://github.com/user-attachments/assets/49564a94-34aa-476b-8bbe-5d94ca619eba" />



<h2>Profundidad de color</h2>
<p>Número de bits por píxel. </p>
<p>Ejemplos: </p>
<ul>
<li>8 bits → 256 colores</li>
<li>24 bits → color verdadero</li>
<li>32 bits → canal alfa</li>
</ul>

<h2>Operaciones sobre bitmaps</h2>
<ul>
<li>Escalado</li>
<li>Rotación</li>
<li>Filtrado</li>
<li>Compresión</li>
<li> Corrección de color</li>
</ul>

<h2>Ventajas</h2>
<ul>
<li>Representación realista</li>
<li>Compatible con fotografías</li>
</ul>

<h2>Desventajas</h2>
<ul>
<li>Archivos pesados</li>
<li>Pierde calidad al escalar</li>
</ul>
