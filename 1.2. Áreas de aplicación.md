<h1>1.2 Áreas de aplicación</h1>
<p> La graficación por computadora tiene un amplio campo de aplicación debido a su capacidad para representar información visual de manera eficiente.
</p>
  <h3>Videojuegos</h3>
<p>Es una de las áreas más visibles. La graficación permite:</p>
<ul>
<li>Modelado de personajes</li>
<li>Escenarios 3D</li>
<li>Efectos visuales</li>
<li>Renderizado en tiempo real</li>
</ul>
<p>Los motores gráficos permiten generar mundos interactivos complejos.</p>

<h3>Cine y animación</h3>
<p>En la industria cinematográfica se utiliza para:</p>
<ul>
<li>Animación 3D</li>
<li>Efectos visuales (VFX)</li>
<li>Composición digital</li>
<li>Simulación de fenómenos físicos</li>
</ul>
<p>Ha permitido crear películas completamente generadas por computadora.</p>

<h3>Diseño asistido por computadora (CAD)</h3>
<p>Se emplea en:</p>
<ul>
<li>Ingeniería mecánica</li>
<li>Arquitectura</li>
<li>Diseño industrial</li>
<li>Manufactura</li>
</ul>
<p>Permite visualizar prototipos antes de fabricarlos.</p>

<h3>Visualización científica</h3>
<p>Se usa para representar datos complejos, por ejemplo:</p>
<ul>
<li>Imágenes médicas</li>
<li>Simulación climática</li>
<li>Modelado molecular</li>
<li>Astronomía</li>
</ul>
<p>Facilita la interpretación de grandes volúmenes de datos.</p>

<h3>Interfaces gráficas de usuario (GUI)</h3>
<p>La graficación es esencial en sistemas operativos y aplicaciones, permitiendo:</p>
<ul>
<li>Ventanas</li>
<li>Iconos</li>
<li>Botones</li>
<li>Animaciones</li>
</ul>
<p>Mejora la interacción humano-computadora.</p>

<h3>Realidad virtual y aumentada</h3>
<p>Permite crear entornos inmersivos usados en:</p>
<ul>
<li>Entrenamiento</li>
<li>Educación</li>
<li>Medicina</li>
<li>Entretenimiento</li>
</ul>
